# EduConnect

**EduConnect** est un site web vitrine et interactif dédié à l'apprentissage du développement web et du design.

---

## Fonctionnalités principales

### Navigation SPA (Single Page Application)

Navigation fluide grâce à un système d'onglets dynamique géré en JavaScript, sans rechargement de page.

### Catalogue de cours

Affichage dynamique de parcours progressifs avec leur niveau, leur duree, les competences travaillees et un projet concret a realiser.

### Quiz interactif

Module de quiz permettant :

* Le calcul automatique du score ;
* L'evaluation de notions HTML, CSS, JavaScript, React et publication web ;
* L'affichage d'explications pedagogiques en fin de quiz ;
* La réinitialisation du questionnaire.

### Présentation de l'équipe

Section dédiée aux formateurs avec une présentation claire et moderne.

### Grille tarifaire

Présentation des différentes offres et abonnements disponibles.

### Formulaire de contact

Formulaire interactif avec :

* Validation des données saisies ;
* Message de confirmation après l'envoi.

### Design responsive

Interface adaptée aux ordinateurs, tablettes et smartphones.

---

## Technologies utilisées

* **HTML5** : Structure sémantique et accessibilité.
* **CSS3** : Mise en page moderne avec Flexbox, Grid, variables CSS (`:root`) et animations.
* **JavaScript (ES6+)** : Gestion dynamique du DOM, navigation SPA et logique du quiz.
* **Font Awesome v6** : Intégration d'icônes vectorielles modernes.

## Exécution du projet

Aucune installation n'est nécessaire.

### Étapes :

1. Télécharger ou cloner le projet.
2. Ouvrir le dossier contenant les fichiers.
3. Double-cliquer sur le fichier `index.html`.

Le site s'ouvrira directement dans votre navigateur web (Chrome, Firefox, Edge ou Safari).

---

## Objectif du projet

EduConnect a pour objectif de proposer une plateforme simple, moderne et performante permettant aux utilisateurs de découvrir les bases du développement web et du design à travers une interface intuitive et interactive.
